# University Work

This repository contains projects, assignments, and learning resources from my Software Engineering studies at the University of Azad Jammu and Kashmir (UAJK). It documents progress, coding practices, and core programming concepts learned during my university learning period.

## Contents
- Projects and assignments (code samples and exercises)
- Notes and learning resources
- Example programs used for practice (e.g., `learn.cpp`)

## Repository Structure
- `learn.cpp` — small example C++ program used while learning
- `clear/` — additional HTML/CSS examples and playground files

## How to Use
1. Browse the folders to find projects or examples you want to review.
2. Open code files with your preferred editor and try building or running them locally.

For example, to compile `learn.cpp` (on Windows with MinGW or similar):

```powershell
g++ learn.cpp -o learn.exe; .\learn.exe
```
